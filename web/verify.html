Verify a shared receipt or audit proof.

Paste JSON from HushPay to validate attached proof bytes when present, recompute bound fields when possible, and inspect only the fields the sender chose to disclose.

This page checks receipt bindings and audit proof payloads. It does not simulate a live validator network or a full reporting product.
What this page checks Proof bytes, receipt bindings, audit proof claims, disclosed values, and whether the payload is internally coherent.
What stays hidden Full wallet history, unrelated counterparties, and any fields the sender did not include in the receipt or audit proof.
What it is not This is not a live validator view, a blockchain explorer, or a full reporting dashboard.
